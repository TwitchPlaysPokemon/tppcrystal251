TwitchPlaysPokémon Anniversary Crystal 251

Holy Bird Jesus! After months of waiting, it's finally out! This page is
where you can download TwitchPlaysPokémon Anniversary Crystal 251,
which was played on stream between February 14 and March 14, 2016.

You will find three links in the Downloads section. Two of them are
.ips files. These are based on vanilla Crystal and Crystal Rev.A,
respectively, and can be applied to a clean copy of vanilla Crystal
using the normal methods. These patches do not interface with the
advanced external AI we showcased on the stream.

The third file is a .zip archive, pokecrystal_ai.zip. When unzipped,
you will find a patch file, an emulator package, and a bunch of Python
and Lua script files. This is a minimal directory structure for run-
ning the AI along with the game.

In the following sections, we will explain how to set up the ROMs and run
the AI.

Setup Instructions

Patching the ROM

  1. Obtain a clean copy of Pokémon Crystal (MD5: 9f2922b235a5eeb78d65594e82ef5dde).
    Due to legal issues, we are unable to provide you with this ROM ourselves.
    However, there are other websites which care little for these issues.
    You can also dump a physical copy of the game.
  2. Create a backup copy of this ROM. You'll need this to apply bugfix patches.
  3. Download Lunar IPS.
  4. Unzip and launch Lunar IPS.
  5. Click "Apply IPS Patch".
  6. Browse to where you downloaded the Crystal 251 IPS file, select it, and click "OK".
  7. Browse to where you saved the clean cipy of Pokémon Crystal, select it, and click "OK".

Setting Up the AI

  1. Download Python 3 for Windows.
  2. Run the installer. Opt to install for all users if you know what you're doing.
  3. Open a Command Prompt window and run "pip install --upgrade pip".
    You may get a massive OSError; ignore it.
    4. NOTE: To open Command Prompt, hold the Win key and press "R".
    In the window that pops up, type "cmd" and hit Enter/Return.
  4. OPTIONAL: In the same Command Prompt window, run "pip install flask".
  5. Download the "AI build + dependencies" pack from the link above.
  6. Extract the files from the ZIP archive.
  7. Patch the ROM using the included .ips file.

Running the AI

  1. Open a Command Prompt window, and navigate to the directory "ai" in the extracted zip file.
    2. To do this, type "cd" and hit the spacebar. In Windows Explorer, copy the path from the address bar.
    If the address does not start with "C:\", type "/d" in the command prompt window, then hit the spacebar.
    Now, paste the copied path into the command prompt window, and hit Enter/Return.
  2. If you ran step 4 of Installing Python, type "python AIServer.py" and hit Enter/Return.
    Otherwise, type "python AIServerStandalone.py" and hit Enter/Return.
  3. Launch the emulator, VBA-RR-243/VBA-RR-243.exe.
  4. Open the newly-patched ROM.
  5. In the emulator, click on Tools > Lua Scripting > New Lua Script Window...
  6. Click the button that says Browse...
  7. If you haven't touched anything in VBA's directory settings, click Open.
    Otherwise, you'll have to navigate to VBA-RR-243/lua first.
  8. Click Run.
  9. Play the game!
